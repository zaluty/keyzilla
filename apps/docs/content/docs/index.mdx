<Accordions>
<Accordion title="What is Keyzilla?">
 
Keyzilla is meant for developers who want to have an easy wazy yet secure to manage their API keys/secrets. 
 
</Accordion>
<Accordion title="Who can use Keyzilla?">
 
Keyzilla can be used in organizations and personal projects too.

</Accordion>
<Accordion title="How does it work?">
 
Keyzilla uses a CLI tool to pull your API keys/secrets from the Keyzilla dashboard and save them to a secret file in the `node_modules` directory of your project.
</Accordion>
<Accordion title="Is it secure?">
keyzilla is secure as it uses `process.env.DATABASE_URL` instead of `k.DATABASE_URL` in production so  each key stays in it environment .
 Learn more about how it works [here](/package/extend)
 </Accordion>
</Accordions>
