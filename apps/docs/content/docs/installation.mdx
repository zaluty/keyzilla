## Installation

To get started with Keyzilla, you need to install it via npm. Run the following command in your project directory:

```bash
npm install keyzilla
```

## Authentication

After installing Keyzilla, the next step is to authenticate with your Keyzilla account. This ensures that you have access to your projects and secrets.
